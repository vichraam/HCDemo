Welcome to HC demo

Below are some of the list of high level concepts used in this demo app.


Technologies


Front End
Angular 10
Web API
.NET Core 3 & Entity Framework - Code first approach
Backend
SQL Lite - self contained database
Unit testing
Xunit
Other components
Bootstrap, Toastr

Concepts


UI Styling
Bootstrap v3
Angular Forms
Template driven form is used for Employee Info form (Adding new employee & editing exisitng employee)
Page Navigation
Angular Routes and RouterModules are used
Server Communication
Used HttpClientModules to communicate to WebAPI for CRUD operations
Search & Sort
@Input is used to pass params from Employee-search component to employee-list component.
Pipe has been used to filter the employee list by search input.
Others
String interpolation, two-way bindings, form validations, observables (mainly in service class), Enabling CORS

Pending Items


Login Form
Api: Implement auth service using Java Web Token with local store
UI: Create login page, store token, create inceptor to pass token for outgoing request (employee)
Mobile Friendly
Update the UI styles to fit mobile screens.
